## [1.0.1] – 2026-03-30

### Added
- Expanded `sitemap.xml` with all pages (homepage, about, services)
- Enhanced structured data (BreadcrumbList schema for navigation)
- Standard project files (README.md at root, CHANGELOG.md, .env.example)
- Comprehensive `.gitignore` with deployment patterns
- Better SEO configuration with lastmod dates on all pages

### Changed
- Updated README.md with latest deployment information

- Improved .gitignore documentation and clarity

### Fixed
- Sitemap now reflects actual site structure with correct paths
- .env.example provides clear configuration reference

## [1.0.0] – 2026-03-28

### Added
- Initial site launch with homepage (`index.html`)
- About page (`pages/about.html`)
- Services page (`pages/services.html`)
- Modern design system with dark theme and gold accents
- Self-hosted fonts (Geist Mono, Lusitana, Lato)
- Custom cursor interactions and scroll animations
- Comprehensive CSS and JavaScript modules
- Basic SEO setup (robots.txt, sitemap.xml, meta tags)
- JSON-LD Person schema markup
- Open Graph social media metadata
- WCAG 2.2 AA accessibility compliance
- Responsive design (mobile-first)
- Deployment to Cloudflare Pages

### Technical Details
- **Framework:** Static HTML, CSS3, JavaScript (no build process)
- **Images:** WebP format, optimised for web
- **Fonts:** Self-hosted (no external font requests)
- **Accessibility:** Semantic HTML, alt text, keyboard navigation, focus indicators
- **Performance:** No external dependencies, minimal HTTP requests
